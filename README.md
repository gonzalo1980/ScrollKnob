# ScrollKnob

Control all Mixxx knobs and faders using your keyboard and mouse wheel — no physical controller needed.

ScrollKnob is a free system that combines AutoHotkey, LoopMIDI and Mixxx to give you fast, ergonomic control over every mixer parameter with just your left hand on the keyboard and your right hand on the mouse.

---

## How it works

Hold down a key and scroll the mouse wheel to adjust any knob or fader in real time — EQ, volume, filter, crossfader, tempo, and effects. Single keypresses handle play, cue, sync and loop.

```
Hold key + scroll wheel → adjust any knob
Single keypress         → play, cue, sync, loop
```

---

## Requirements

- Windows
- [Mixxx 2.5](https://mixxx.org) — free DJ software
- [AutoHotkey v2](https://www.autohotkey.com) — free
- [LoopMIDI](https://www.tobias-erichsen.de/software/loopmidi.html) — free

## Control map

| Key + wheel | Control |
|---|---|
| `S` + wheel | Left deck volume |
| `G` + wheel | Right deck volume |
| `3` + wheel | EQ High left deck |
| `E` + wheel | EQ Mid left deck |
| `D` + wheel | EQ Low left deck |
| `4` + wheel | EQ High right deck |
| `R` + wheel | EQ Mid right deck |
| `F` + wheel | EQ Low right deck |
| `C` + wheel | Filter left deck |
| `V` + wheel | Filter right deck |
| `X` + wheel | Crossfader |
| `A` + wheel | Tempo left deck |
| `H` + wheel | Tempo right deck |
| `F1`–`F3` + wheel | FX knobs left deck |
| `F4` + wheel | Combined FX volume left deck |
| `F5`–`F7` + wheel | FX knobs right deck |
| `F8` + wheel | Combined FX volume right deck |

| Key | Action |
|---|---|
| `W` | Play / Pause left deck |
| `T` | Play / Pause right deck |
| `2` | Cue left deck |
| `5` | Cue right deck |
| `1` | BPM Sync left deck |
| `6` | BPM Sync right deck |
| `Q` | Loop left deck |
| `Y` | Loop right deck |
| `F9` | Enable / Disable ScrollKnob |
